This project is an AI-powered agent that converts natural language questions into SQL queries and executes them on a SQLite database. It then returns the results in simple English.

Built using an agentic workflow, it demonstrates how LLMs can be used to interact with structured data in real time.

🚀 Features
Ask questions in plain English
Automatically generates valid SQL queries
Executes queries on a SQLite database
Retries and fixes broken SQL using LLM feedback loop
Converts SQL results into human-readable answers
Interactive web UI using Streamlit
